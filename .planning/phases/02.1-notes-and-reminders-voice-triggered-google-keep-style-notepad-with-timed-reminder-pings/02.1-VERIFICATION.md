## Database Schema Verification

**Table: `notes`**

```sql
CREATE TABLE IF NOT EXISTS notes (
    id TEXT PRIMARY KEY,
    content TEXT NOT NULL,
    note_type TEXT DEFAULT 'note',       -- 'note' | 'reminder'
    due_at TEXT,                         -- ISO8601 UTC, null for plain notes
    color TEXT DEFAULT 'default',        -- color key: default, deep-blue, dark-teal, amber-night, rose-night, mauve-glass
    pinned INTEGER DEFAULT 0,            -- 0 | 1
    dismissed INTEGER DEFAULT 0,         -- 0 = active, 1 = dismissed (for reminders)
    created_at TEXT NOT NULL,
    updated_at TEXT NOT NULL
)
```

**Index:**
```sql
CREATE INDEX IF NOT EXISTS idx_notes_due_active
ON notes(due_at, dismissed)
```

All columns present and correctly typed. Index present for due-reminder filtering.
