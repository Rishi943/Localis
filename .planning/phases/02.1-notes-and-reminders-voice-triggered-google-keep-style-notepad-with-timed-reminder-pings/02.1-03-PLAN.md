Then generate a short chime using Python's built-in capabilities. The chime must be:
- Duration: ~0.5 seconds
- Tone: 880 Hz sine wave (A5 note — pleasant, not jarring)
- Format: WAV first (Python stdlib), then convert if ffmpeg available, otherwise use WAV renamed as fallback

Use this Python script to generate it:
```python
import struct
import math
import wave

sample_rate = 44100
duration = 0.5  # seconds
frequency = 880  # Hz (A5 — gentle, pleasant)
volume = 0.3    # 30% volume — subtle

num_samples = int(sample_rate * duration)
samples = []
for i in range(num_samples):
    t = i / sample_rate
    # Sine wave with exponential decay envelope for natural chime feel
    envelope = math.exp(-4 * t / duration)
    sample = volume * envelope * math.sin(2 * math.pi * frequency * t)
    samples.append(int(sample * 32767))

# Write as WAV
with wave.open('/home/rishi/Rishi/AI/Localis/app/static/sounds/chime.wav', 'w') as wav_file:
    wav_file.setnchannels(1)
    wav_file.setsampwidth(2)
    wav_file.setframerate(sample_rate)
    wav_file.writeframes(struct.pack('<' + 'h' * len(samples), *samples))
print('chime.wav generated')
```

Then attempt to convert to MP3 using ffmpeg:
```bash
ffmpeg -i /home/rishi/Rishi/AI/Localis/app/static/sounds/chime.wav \
       -codec:a libmp3lame -qscale:a 2 \
       /home/rishi/Rishi/AI/Localis/app/static/sounds/chime.mp3 -y 2>/dev/null \
  && echo "MP3 generated" \
  || cp /home/rishi/Rishi/AI/Localis/app/static/sounds/chime.wav \
        /home/rishi/Rishi/AI/Localis/app/static/sounds/chime.mp3 \
  && echo "WAV copied as chime.mp3 fallback"
```
